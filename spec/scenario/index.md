# 利用シナリオ一覧

- 管理者用メモの宣言的設定と現在のフォーム設定の差分を確認する
- マイグレーションを実行してフォーム設定を更新する
- 管理者用メモの設定でフォーム設定を強制的に上書きする
- 稼働中のフォーム設定から宣言的な構成を生成して管理者用メモに反映する
- フォーム設定をリセットして全カスタムフィールドを削除する
- 複数アプリを依存順に一括diff/migrate/override/capture/dumpする（Multi-Appモード）
- シードデータファイルのレコードをkintoneアプリにUpsert方式で反映する
- kintoneアプリの全レコードをキャプチャしてシードデータファイルに保存する
- 複数アプリのシードデータを依存順に一括反映/キャプチャする
