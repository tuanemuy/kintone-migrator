# 利用シナリオ一覧

## フォームスキーマ管理

- 管理者用メモの宣言的設定と現在のフォーム設定の差分を確認する
- マイグレーションを実行してフォーム設定を更新する
- 管理者用メモの設定でフォーム設定を強制的に上書きする
- 稼働中のフォーム設定から宣言的な構成を生成して管理者用メモに反映する
- フォーム設定をリセットして全カスタムフィールドを削除する

## データ・カスタマイズ管理

- シードデータファイルのレコードをkintoneアプリにUpsert方式で反映する
- kintoneアプリの全レコードをキャプチャしてシードデータファイルに保存する
- JS/CSSカスタマイズをYAML設定ファイルに基づいて適用する

## アクセス権管理

- フィールドアクセス権をYAML設定ファイルに基づいて適用する
- kintoneアプリの現在のフィールドアクセス権をキャプチャしてYAMLファイルに保存する
- アプリのアクセス権をYAML設定ファイルに基づいて適用する
- アプリのアクセス権をキャプチャしてYAMLファイルに保存する
- レコードのアクセス権をYAML設定ファイルに基づいて適用する
- レコードのアクセス権をキャプチャしてYAMLファイルに保存する

## アプリ設定管理

- 一覧（ビュー）設定をYAML設定ファイルに基づいて適用する
- 一覧（ビュー）設定をキャプチャしてYAMLファイルに保存する
- プロセス管理（ワークフロー）設定をYAML設定ファイルに基づいて適用する
- プロセス管理設定をキャプチャしてYAMLファイルに保存する
- 一般設定（アプリ名・テーマ・アイコン等）をYAML設定ファイルに基づいて適用する
- 一般設定をキャプチャしてYAMLファイルに保存する
- 通知設定（一般通知・レコード条件通知・リマインダー）をYAML設定ファイルに基づいて適用する
- 通知設定をキャプチャしてYAMLファイルに保存する
- グラフ・レポート設定をYAML設定ファイルに基づいて適用する
- グラフ・レポート設定をキャプチャしてYAMLファイルに保存する
- アクション設定をYAML設定ファイルに基づいて適用する
- アクション設定をキャプチャしてYAMLファイルに保存する
- アプリ管理者用メモをYAML設定ファイルに基づいて適用する
- アプリ管理者用メモをキャプチャしてYAMLファイルに保存する
- プラグイン設定をYAML設定ファイルに基づいて適用する
- プラグイン設定をキャプチャしてYAMLファイルに保存する

## マルチアプリ管理

- 複数アプリを依存順に一括diff/migrate/override/capture/dumpする（Multi-Appモード）
- 複数アプリのシードデータを依存順に一括反映/キャプチャする
- 複数アプリの各種設定を依存順に一括apply/captureする
