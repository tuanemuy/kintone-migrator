# アダプターの実装を行う

## 背景

- `spec/index.md` に設計を行った
- `docs/backend_implementation_example.md` にコアアーキテクチャの実装例を記載した
- `docs/kintone.md` にkintoneのドキュメントを列挙した
- `spec/database.md` にデータベース設計を記載した

## タスク

1. 未実装のアダプターを確認する
2. 設計に従ってアダプターの実装を続ける
3. 型チェックとリンターを実行し、問題がなくなるまで修正を続ける

## 条件

- リポジトリのアダプターとして、今回は `drizzleSqlite` を使う
- その他のアダプターは適切なプロバイダーを検討して、そのディレクトリ内に実装する
