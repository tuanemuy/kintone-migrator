# ADR — Issue #128

## ADR-001: init コマンドの認知的複雑度を解消するためにヘルパー関数を抽出

**日付:** 2026-03-26
**状態:** 採用

### コンテキスト

`space-id` の分岐ロジック（`if (spaceId) { ... } else { ... }`）を追加した結果、`run` 関数の認知的複雑度が Biome の上限 25 を超えて 28 に達した。

### 決定

以下の2つのヘルパー関数を `init.ts` 内に抽出し、`run` 関数の複雑度を削減した:

- `printDryRunPreview()`: dry-run 時のプレビュー表示ロジック
- `captureApps()`: 各アプリに対する capture ループ

### 理由

- lint 警告を解消しつつ、既存のテストに影響を与えない
- ヘルパー関数はモジュール内プライベートであり、外部 API への変更は不要
- 既存コードのロジックを変更せず、構造的なリファクタリングのみ
